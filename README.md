# Airbnb Price Analysis

## Objective
Analyze Airbnb listing prices and identify patterns related to location, room type, and availability.

## Key Insights
- Prices are right-skewed with most listings in lower ranges
- Location appears to influence pricing
- Entire homes are generally more expensive
- No strong relationship between price and availability

## Tools
- Python
- Pandas
- Matplotlib

## Dataset
Airbnb dataset sourced from Inside Airbnb.
